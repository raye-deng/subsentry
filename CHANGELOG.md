## [0.1.0] - 2026-02-04

### Added
- Initial MVP release
- Landing page with product introduction
- Dashboard with subscription management
- Mock data for demonstration
- Duplicate subscription detection UI
- Savings calculator
- Interactive subscription table
- Responsive design for all devices
- GitHub integration

### Features
- **Landing Page**: Product introduction, value proposition, social proof
- **Dashboard**: View all subscriptions with analytics
- **Duplicate Detection**: Identify overlapping tools across teams
- **Savings Calculator**: Show potential savings from consolidation
- **Interactive UI**: Cancel and consolidate subscriptions (mock)
- **Stats Grid**: 4 key metrics (Monthly spending, Duplicates, Zombie, Potential)

### Technical
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (icons)
- Recharts (charts)
- Responsive design
