# Screenshots

2. **dashboard.png** - The main dashboard showing subscriptions and stats
3. **duplicates.png** - Duplicate subscription detection view
4. **mobile-dashboard.png** - Dashboard on mobile device

## How to Take Screenshots

1. Open the local development server: `npm run dev`
2. Navigate to the desired page
3. Take a screenshot at 1920x1080 resolution
4. Save as PNG in this directory

## Tools

- macOS: Cmd + Shift + 4
- Windows: Windows Key + Shift + S
- Browser DevTools: Toggle device toolbar for mobile screenshots
